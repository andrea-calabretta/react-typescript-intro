Installare Node.js:
https://nodejs.org/en/

Installare TypeScript:
npm install -g typescript

Creare una nuova cartella e inizializzarla per il progetto con npm:
npm init -y
Questo comando crea un file package.json con le informazioni di base del progetto.

Inizializzare un progetto TypeScript nella stessa cartella:
tsc --init

Questo comando crea un file tsconfig.json, che Ã¨ il cuore della configurazione TypeScript.
Qui si possono personalizzare le impostazioni come target di compilazione, percorsi dei file, opzioni del compilatore, ecc.
